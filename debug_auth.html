Debug Authentication Helper

This page helps set up authentication for the dashboard.

Instructions:

  1. Click "Set Debug Token" button above
  2. Open http://localhost:3000 in another tab
  3. The dashboard should now display data without requiring login
  4. If you need to login again, you can also use any email/password on the login page

Note: This is for development/testing purposes only. The actual API endpoints don't require authentication for testing.
